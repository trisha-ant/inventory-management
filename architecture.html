Request Data Flow

1 User Input Filter dropdown bound to reactive ref
2 useFilters Composable emits current filter state
3 api.js Builds query string, axios GET request
4 FastAPI apply_filters() on in-memory list
5 Pydantic Validates response shape
6 Vue Render computed() transforms → template
